Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install enlumin-cli

NPM

$ npm install enlumin-cli

Yarn

$ yarn install enlumin-cli
